### 4.18

优化了下聊天，加了个工具调用返回，目前可以根据会话ID获取聊天记录，以及添加了会话列表，没有绑定外键，因为我发现如果绑定外键的话，有些地方会很麻烦，
比如说，新会话，需要生成一个标题，如何插入到表中，这个时候如果同步去生成，肯定是不行的，太慢了，肯定要异步，那先插入列表，
生成之后再进行更新，这样子确实可以，但是很明显的是多了一步操作，而且要依赖一个完全没必要的service/mapper，有点得不偿失的感觉。
不过虽然写的是异步，但是生成标题还是非常快的！太强了
